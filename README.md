# HCBS-utils

**Utility** library to manipulate process scheduling policies, CPU affinities and cgroups (with focus on the [HCBS](https://github.com/Yurand2000/HCBS-patch) patchset).

The code was partially extracted from [HCBS-Test-Suite](https://github.com/Yurand2000/HCBS-Test-Suite) to be used also in other crates.
